# Smart Login System

## 1.Description
A simple authentication system built as a test project. It includes user registration, login functionality, and navigation to a home page after successful login.


## 2.Live Demo
[Live demo](https://doniamagdy.github.io/Smart-Login/)


## 3.Features

-  User Registration
-  User Login
-  Redirect to Home Page after login


## 4.Built With

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla JS)


## 5.Project Description

This project was created as a simple test task. The main goal was to implement basic authentication flow without advanced backend features.

The application allows users to:
1. Register by entering their details
2. Log in using registered credentials
3. Get redirected to a home page after successful login

The home page is a simple placeholder page with no additional functionality.
